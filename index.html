How it works
01.
Copy this template on GitHub

Click Use this template above, then Create a new repository. Give it a name and click Create repository. This makes a fresh copy in your GitHub account — no history carried over.

02.
Get the files onto your computer

Install GitHub Desktop. On your new repo page, click the green Code button, then Open with GitHub Desktop. Choose where to save it and click Clone.

03.
Open the project in Cursor

Install Cursor — a free AI code editor. Open it, go to File → Open Folder, and select the folder you just cloned.

04.
Open the AI and start building

Press ⌘I (Mac) or Ctrl I (Windows) to open Cursor's AI. Cursor reads the project automatically — just describe what you want and it handles the rest.
