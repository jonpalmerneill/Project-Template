What's included
Vite Local preview server. See changes instantly.
Vercel Auto-deploys when you push to GitHub.
Supabase Database and user authentication, optional.
Password gate Optional shared-password protection.
Alpine.js Reactive tabs, filters, and live data in HTML — no framework needed.
Motion Animation library, pre-installed.
Figma Your AI editor can pull designs and tokens directly from your files.
GSAP Advanced scroll animations, timelines, and pinned sections.
Three.js 3D scenes, WebGL backgrounds, and 3D model loading.
PixiJS 2D WebGL, sprites, particle systems, and canvas effects.
Lenis Smooth momentum scrolling.
Swiper Touch-friendly carousels and sliders.
Page loader Animated progress bar on entry.
Page transitions Animated fade between pages.
Custom cursor Styled cursor with hover states.
GLSL shaders Custom visual effects — CRT, distortion, noise.
Contact form Email submissions via Formspree or Resend.
Stripe Payments via serverless checkout.
Chart.js Data visualization and charts.
Dot field Interactive canvas dot background.
Card tilt 3D perspective tilt on mouse move.
Text scramble Characters cycle randomly before settling.
Leaflet maps Interactive maps via OpenStreetMap. No API key needed.
Mapbox GL 3D terrain, building extrusion, and satellite imagery with a free API key.
Airtable Use a spreadsheet as a database for content-driven prototypes.
External data APIs Weather, news, government data, and more — free tiers included.
Surveys Multi-step forms with branching logic via SurveyJS or Supabase.
Audio player Custom play/pause, scrubbing timeline, and multi-track support.
Dark / light mode Theme toggle with system preference detection and no flash on load.
How it works
01.
Copy this template on GitHub

Click Use this template above, then Create a new repository. Give it a name and click Create repository.

02.
Get the files onto your computer

Install GitHub Desktop. On your new repo page, click the green Code button, then Open with GitHub Desktop. Choose where to save it and click Clone.

03.
Open the project in Cursor

Install Cursor — a free AI code editor. Open it, go to File → Open Folder, and select the folder you just cloned.

04.
Open the AI and start building

Press ⌘I (Mac) or Ctrl I (Windows) to open Cursor's AI. Cursor reads the project automatically — just describe what you want and it handles the rest.
