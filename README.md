# Starter Template

A minimal website starter you customize and deploy by chatting with Claude Code — no terminal commands needed.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/jonpalmerneill/Project-Template)

---

## What's included

- **Local dev server** — see changes in your browser instantly
- **Password gate** — optional shared-password protection for private sites
- **Supabase** — optional database and user authentication
- **Motion** — animation library, ready to use
- **Vercel** — deploys automatically when you push to GitHub
- **Figma** — Claude can pull designs and tokens directly from your Figma files

## Get started

### Step 1 — Use this template

Click **Use this template → Create a new repository** at the top of this page on GitHub.

Give your project a name, set it to **Private**, and click **Create repository**.

### Step 2 — Get the files on your computer

**No terminal:** Install [GitHub Desktop](https://desktop.github.com), then on your new repo page click **Code → Open with GitHub Desktop**. Choose where to save it and click Clone.

**Terminal:** `git clone https://github.com/your-username/your-repo-name.git`

### Step 3 — Open in your AI editor

**Claude Code** (recommended — reads this project's setup automatically)

1. Install Claude Code: [claude.ai/code](https://claude.ai/code)
2. Open GitHub Desktop → click **Repository → Open in Terminal**
3. Type `claude` and press Enter

That's the one terminal command you'll need. From here, Claude handles everything — just talk to it.

> Using VS Code? Install the [Claude Code extension](https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code) and open the terminal from within VS Code instead.

**Cursor** (alternative — good if you prefer a visual editor)

1. Install Cursor: [cursor.com](https://cursor.com)
2. Open Cursor → **File → Open Folder** → select your project folder
3. Use the chat panel to start building

> Cursor doesn't read `CLAUDE.md` automatically. For best results, open `CLAUDE.md` and paste its contents into your first chat message as context.

## What to ask Claude

**First time setup:**
- "Get this running so I can preview it"
- "Set up GitHub and deploy to Vercel"
- "Connect Supabase"
- "Turn on password protection"

**Designing:**
- "Change the colors to dark mode"
- "Use [font name] from Google Fonts"
- "Here's my Figma file: [URL] — implement this design"
- "Sync my Figma color tokens to the stylesheet"

**Building:**
- "Add a contact form"
- "Add a sign-up / login page"
- "Add a section that shows data from my database"
- "Add a 3D background using Three.js"
- "Add a chart showing [your data]"
- "Embed my Calendly / Typeform / YouTube video"

**Shipping:**
- "Push my changes and deploy"
