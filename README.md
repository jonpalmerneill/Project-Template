# Starter Template

A minimal website starter you customize and deploy by chatting with AI — no terminal commands needed.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/jonpalmerneill/Project-Template)

---

## What's included

- **Local dev server** — see changes in your browser instantly
- **Password gate** — optional shared-password protection for private sites
- **Supabase** — optional database and user authentication
- **Motion** — animation library, ready to use
- **Vercel** — deploys automatically when you push to GitHub
- **Figma** — your AI editor can pull designs and tokens directly from your files

---

## Get started

### Step 1 — Use this template

Click **Use this template → Create a new repository** at the top of this page on GitHub.

Give your project a name, set it to **Private**, and click **Create repository**.

### Step 2 — Get the files on your computer

Install [GitHub Desktop](https://desktop.github.com), then on your new repo page click **Code → Open with GitHub Desktop**. Choose where to save it and click Clone.

### Step 3 — Open in Cursor

[Cursor](https://cursor.com) is a free AI code editor. Open it, go to **File → Open Folder**, and select your project folder.

The project is pre-configured so Cursor understands the entire setup automatically. Just start chatting — Cursor handles installing dependencies, starting the preview server, setting up Vercel, and making whatever changes you ask for.

## What to ask

**First time setup:**
- "Get this running so I can preview it"
- "Set up GitHub and deploy to Vercel"
- "Connect Supabase"
- "Turn on password protection"

**Designing:**
- "Change the colors to dark mode"
- "Use [font name] from Google Fonts"
- "Here's my Figma file: [URL] — implement this design"
- "Sync my Figma color tokens to the stylesheet"

**Building:**
- "Add a contact form"
- "Add a sign-up / login page"
- "Add a section that shows data from my database"
- "Add a 3D background"
- "Embed my Calendly / Typeform / YouTube video"

**Shipping:**
- "Push my changes and deploy"

## Accounts you'll need

All free tiers are sufficient to start.

| Account | What it's for |
|---------|--------------|
| [GitHub](https://github.com) | Stores your code |
| [GitHub Desktop](https://desktop.github.com) | Get files onto your computer without a terminal |
| [Cursor](https://cursor.com) | AI editor that builds and deploys for you |
| [Vercel](https://vercel.com) | Hosts your site |
| [Supabase](https://supabase.com) | Database and user auth (only if you need it) |
